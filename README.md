## 📂 Featured Projects

### 📊 [Sales Dashboard — Brazil E-commerce](https://github.com/Miich2000/Dashboard_streamlit)
Interactive dashboard built with Streamlit and Plotly, consuming real-time data from a REST API.  
Analyzes revenue, sales volume and seller performance with dynamic filters by region, year and seller.  
🌐 **Live demo:** [dashboardapp-strmltvendasmiich2000.streamlit.app](https://dashboardapp-strmltvendasmiich2000.streamlit.app/)

---

### 🤖 [ML Classification — Bank Marketing](https://github.com/Miich2000)
End-to-end machine learning pipeline to predict customer investment product adoption.  
Covers preprocessing (OneHotEncoder, MinMaxScaler), model comparison (Decision Tree, KNN, Dummy Classifier), evaluation with confusion matrix and model serialization with Pickle.

---

### 🌍 [Canada Immigration Analysis (1980–2013)](https://github.com/Miich2000/canada-immigration-analysis)
Exploratory data analysis of UN immigration data. Identifies temporal patterns, continental shifts and geopolitical context behind migration flows.  
Combines static (Seaborn) and interactive (Plotly) visualizations with data storytelling.

---

### 🎌 [Anime Popularity Analysis](https://github.com/Miich2000/projeto-animes-popularidade)
Data analysis project combining Python and SQL to explore anime popularity patterns.  
Integrates relational database queries with exploratory analysis in Python.
